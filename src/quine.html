Get Started with the TaskPaper format: TaskPaper knows about four things: projects, tasks, notes, and tags. - To create a task, type a line starting witha dash followed by a space. - To create a project, type a line ending with a colon. - Any line that isn't a task or project is a note. - To create a tag, type '@' followed by a name anywhere in a project, task, or note. You use these basic parts in any way you like. TaskPaper doesn't force a particular system on you; it provides the basic to-do list elements and then you use them as you see fit.
